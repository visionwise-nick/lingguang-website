# Lingguang 官方网站

这是 Lingguang (灵光) 的官方网站，展示 AI 驱动的 App 生成器的功能和特性。

## 网站结构

```
docs/
├── index.html          # 主页
├── privacy.html        # 隐私政策
├── terms.html          # 用户协议
├── styles.css          # 样式文件 (暗色主题)
├── script.js           # JavaScript功能 (粒子效果等)
├── assets/
│   └── icon/
│       └── app_icon.png # 应用图标
└── README.md           # 说明文档
```

## 功能特性

### 主页 (index.html)
- 响应式暗色科技风设计
- 产品介绍：自然语言生成 App
- 核心功能：零代码、即时生成、跨平台、易分享
- 下载链接区域
- 粒子动画和光效

### 隐私政策 (privacy.html)
- 针对 AI 生成内容的数据隐私说明
- 用户输入提示词的处理说明

### 用户协议 (terms.html)
- AI 生成服务的免责声明
- 积分系统说明

## 部署说明

### GitHub Pages 部署

1. 提交更改到 GitHub 仓库
2. 进入仓库 Settings -> Pages
3. Source 选择 "Deploy from a branch"
4. Branch 选择 "main" (或 master)，文件夹选择 "/docs"
5. 点击 Save

网站将上线于：`https://visionwise-nick.github.io/Selfcode/` (或其他自定义域名)
